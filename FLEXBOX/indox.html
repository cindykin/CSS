Flex Container (Parent)

property yang bisa dipakai :

  • display: flexbox | inline-flex;
    = memberitahu kita akan pake modul flexbox
  • flex-direction : row | row-reverse | column | column-reverse;
    = mengontrol bagaimana item didalam container ditampilkan
  • flex-wrap: nowrap | wrap | wrap-reverse;
    = mengubah item jadi lebih dari 1 baris. Flexbox secara default akan mencoba memasukkan semua elemen ke dalam satu baris
  • flex-flow: <‘flex-direction’> || <‘flex-wrap’>
    = shorthand flex-direction & flex-wrap
  • justify-content: flex-start | flex-end | center | space-between | space-around;
    = menyelaraskan/align item didalam container di sepanjang main axis
  • align-items: flex-start | flex-end | center | baseline | stretch;
    = menyelaraskan item di sepanjang cross axis
  • align-content: flex-start | flex-end | center | space-between | space-around | stretch;
    = menyelaraskan item yg punya lebih dr 1 baris


Flex Item (Child)

property yg bisa dipakai :

  • order: [integer];
    = mengontrol urutan kemunculannya dalam container flex
  • flex-grow: [number]; /* default 0 */
    = mengontrol urutan kemunculannya dalam wadah fleksibel
  • flex-shrink: [number]; /* default 1 */
    = menentukan kemampuan item fleksibel untuk menyusut jika perlu
  • flex-basis: [length] | auto; /* default auto */
    = menentukan ukuran default elemen sebelum ruang yang tersisa didistribusikan
  • flex: none | [ < 'flex-grow'> < 'flex-shrink'> || < 'flex-basis'> ]
    = shorthand flex-grow, flex-shrink, flex-basis
  • align-self: auto | flex-start | flex-end | center | baseline | stretch;
    = memungkinkan perataan default (atau yang ditentukan oleh align-items) untuk diganti untuk item flex individual
