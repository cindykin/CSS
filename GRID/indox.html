Terminology

KOLOM/HORIZONTAL = manjang kesamping || BARIS/VERTIKAL = manjang kebawah
  • grid container = elemen pembungkus grid
  • grid item = elemen yang berada didalam grid container
  • grid line = garis horizontal/vertikal yang memisahkan grid menjadi beberapa bagian ditandai dengan angka
  • grid cell = perpotongan antara baris dan kolom didalam grid
  • grid area = kumpulan lebih dari satu grid cell yang membentuk kotak
  • grid track = ukuran/jarak antara 2 grid line(ukuran cell), bisa kolom/baris
  • grid gap = jarak antar cell/grid track
