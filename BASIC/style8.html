CSS Combinator Selectors

BELOM SELESAI BRO

keyword : Combinator Selectors di css


Kombinator adalah sesuatu yang menjelaskan hubungan antara Selector

  1. Descendant(keturunan) Selector

    2. untuk memilih elemen turunan pake (spasi) aja

    <!DOCTYPE html>
    <html>
    <head>
    <style>
    div p {
      color: maroon;
    }
    </style>
    </head>
    <body>
    <div>
      <p> hai hai </p>
    </div>
    </body>
    </html>

  2. Child Selector

    3. untuk memilih elemen yg merupakan anak dari elemen, pake (>) aja

    <!DOCTYPE html>
    <html>
    <head>

    <style>
    div > p {
      color: maroon;
      margin-left: 40px;
    }
    </style>
    </head>
    <body>
    <p>This is a paragraph.</p>
    </body>
    </html>

  3. Adjacent Sibling (Saudara yang Berdekatan) Selector

    4. biasanya digunakan kalo setiap elemen punya gaya yg beda

    menggunakan bantuan attr style yg valuenya berisi property css

    <!DOCTYPE html>
    <html>
    <body>

    <h1 style="color:blue;text-align:center;">This is a heading</h1>
    <p style="color:red;">This is a paragraph.</p>

    </body>
    </html>

  4. General Sibling (Saudara Umum) Selector

    5. biasanya digunakan kalo setiap elemen punya gaya yg beda

    menggunakan bantuan attr style yg valuenya berisi property css

    <!DOCTYPE html>
    <html>
    <body>

    <h1 style="color:blue;text-align:center;">This is a heading</h1>
    <p style="color:red;">This is a paragraph.</p>

    </body>
    </html>
