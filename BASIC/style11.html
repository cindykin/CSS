CSS Linear Gradient

keyword : linear gradient

  • untuk membuat gradient

  • value direction :
    • Top to Bottom (this is default) - datang dari atas ke bawah
    • Left to Right - datang dari kiri ke kanan
    • Diagonal

  • cara pakai :

    • div {
    background-image: linear-gradient(180deg, red, yellow);
    background-image: linear-gradient(direction/angle, color-stop1, color-stop2, ...);
    }
