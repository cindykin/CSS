Cara-cara Menambah/Menghubungkan CSS

keyword : konek css

  1. Eksternal CSS

    2. ditaruh di head, menggunakan tag link.

    link file cssnya jadi valuenya href

    <!DOCTYPE html>
    <html>
    <head>

    <link rel="stylesheet" href="mystyle.css"> <--- INI NEHH

    </head>
    <body>
    <h1>This is a heading</h1>
    </body>
    </html>

  2. Internal CSS

    3. biasanya digunakan kalo setiap hal punya gaya yg beda

    <!DOCTYPE html>
    <html>
    <head>

    <style> <--- MULAI DARI OPEN TAG GEYS
    h1 {
      color: maroon;
      margin-left: 40px;
    }
    </style>     <--- SAMPE CLOSE TAG SINI

    </head>
    <body>
    <p>This is a paragraph.</p>
    </body>
    </html>

  3. Inline CSS

    4. biasanya digunakan kalo setiap elemen punya gaya yg beda

    menggunakan bantuan attr style yg valuenya berisi property css

    <!DOCTYPE html>
    <html>
    <body>

    <h1 style="color:blue;text-align:center;">This is a heading</h1>
    <p style="color:red;">This is a paragraph.</p>

    </body>
    </html>

    Cascading Order

    style model apa yg akan dipake jika ada lebih dari 1 cara penghubungan yg dipake utk sebuah elemen html?
    Semua style di halaman akan "bertingkat" menjadi lembar style "virtual" baru, dgn no 1 punya prioritas tertinggi:
    1. Inline css (inside an HTML element)
    2. External and internal style sheets (in the head section)
    3. Browser default
    Inline css punya prioritas paling tinggi, jadi style yg diterapkan di Inline style akan menimpa Eksternal style & Browser default
