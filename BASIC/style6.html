CSS Box Model

keyword : box model css

  • Dalam css menganggap ada kotak yang membungkus setiap elemen HTML. Terdiri dari: margin, border, padding, dan konten sebenarnya

  • penjelasan bagian :
    • Content - The content of the box, where text and images appear
    • Padding - Clears an area around the content. The padding is transparent
    • Border - A border that goes around the padding and content
    • Margin - Clears an area outside the border. The margin is transparent

  • Ukuran total elemen harus dihitung seperti ini:
    • Panjang elemen total = width + left padding + right padding + left border + right border + left margin + right margin
    • Tinggi elemen total = height + top padding + bottom padding + top border + bottom border + top margin + bottom margin
contoh :

width : 500px
height : 350px
margin : 15px
border : 10px
padding : 5px

hasilnya
panjang/width elemen total = 500 + 5 + 5 + 10 + 10 + 15 + 15 = 560px
tinggi/height elemen total = 350 + 5 + 5 + 10 + 10 + 15 + 15 = 410px
