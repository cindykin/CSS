CSS Syntax

keyword : stuktur css

  • 1 set css terdiri dari 1/lebih selector dan 1 blok deklarasi

  • si selector menargetkan ke elemen yang ingin kamu style

  • setiap deklarasi didalam blok deklarasi harus menyertakan nama properti css serta titik dua lalu diikuti nilainya, dan harus dipisahkan dengan titik koma

  • si blok deklarasi harus dikelilingi dengan kurung kurawal serta dipisahkan satu sama lain dengan titik koma
