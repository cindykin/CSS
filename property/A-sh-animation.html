SHORT HAND-animation

  • shorthand untuk : (tak apa2 jika salah satu nilai dibwh hilang, selama masih sesuai urutan)
    • animation-name (required) - nama keyframe utk mengaitkannya ke properti animation
    • animation-duration (required) - durasi animasi
    • animation-timing-function - menentukan kurva dari kecepatan animasi, memungkinkan animasi untuk mengubah kecepatan selama durasinya
    • animation-delay - menentukan penundaan untuk memulai animasi, klo negatif animasi akan dimulai seolah-olah telah diputar selama N detik
    • animation-iteration-count - menentukan berapa kali animasi harus dijalankan
    • animation-direction - nentukan apakah animasi harus dimainkan maju, mundur, atau dalam siklus alternatif
    • animation-fill-mode - nentukan gaya untuk elemen target saat animasi tidak diputar (sebelum dimulai, setelah berakhir, atau keduanya)
    • animation-play-state - menentukan apa animasi harus dijalankan/dipause
    • initial - mengubahnya ke nilai default
    • inherit - mewarisi nilai property parent elemennya
  • cara pakai :

        div {
            animation: hehe 5s linear 2s infinite alternate;
        }
