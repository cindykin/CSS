background-size

  • digunakan utk memspesifikan ukuran bg image
  • valuenya :
    • auto(default) - bg image ditampili dlm original size
    • length - nentukan width & height dari bg image. value 1 = width, value 2 = height, jika cuma 1 yg diketik value 2 = auto
    • % - nentukan width & height dgn % dari size parent elemennya
    • cover - resize bg image utk mencover seluruh container biasa distretch
    • contain - resize image utk mempastikan image terlihat. Jadi bila img lebih kecil drpd kontainer jadinya sisa kosongnya warna putih
    • initial - mengubahnya ke nilai default
    • inherit - mewarisi nilai property parent elemennya
  • cara pakai : 

        body {
            background-size: 300px 500px;
        }
