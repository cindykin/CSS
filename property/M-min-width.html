min-width

  • jika konten lebih kecil dari nilai min-width, nilai min-width akan diterapkan
  • jika layar lebih besar dari nilai min-width, nilai min-width tidak dipake
  • kayak width biasa guys ga ngefek
  • valuenya :
    • length - nilai angka dlm satuan px, cm, em, dll
    • initial - menjadikan lebar/tinggi ke nilai default
    • inherit - Tinggi / lebar akan diwarisi dari nilai induknya
  • cara pakai :

        div {
            min-width: 500px;
        }
contoh :
min-width : 500px;
maka saat dibuka dalam ukuran layar yg kurang dari 500px, width elemen tersebut akan tetap 500px
