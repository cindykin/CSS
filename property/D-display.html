display

  • digunakan untuk menentukan perilaku display (jenis dari rendering box) dari sebuah elemen
  • value :
    • inline - tidak bisa diatur sizenya, ukuran elemen pas dengan isinya
    • block - bisa atur size, enter gila, width memenuhi lebar halaman
    • inline-block - bisa atur size, width memenuhi lebar halaman
    • none - menyembunyikan elemen tanpa menghapus, tempatnya menjadi hilang (kalo visibility : hidden membuat elemen hilang, tapi tempatnya masih ada)

    • flex/grid - menampilkan elemen jadi block-level flex/grid container
    • inline flex/grid - menampikan elemen jadi inline-level flex/grid container
    • initial - mengubahnya ke nilai default
    • inherit - mewarisi nilai property parent elemennya
  • cara pakai :

        body {
            display: grid;
        }
