text-align

  • digunakan untuk mengatur alignment horizontal teks yg berada didalam sebuah elemen
  • suatu teks bisa diratakan kiri atau kanan, di tengah, atau dirata kanan-kiri
  • value :
    • left - rata kiri
    • right - rata kanan
    • center - rata tengah
    • justify - rata kanan-kiri
    • initial - mengubahnya ke nilai default
    • inherit - mewarisi nilai property parent elemennya
  • cara pakai :

        body {
            text-align: justify;
        }
