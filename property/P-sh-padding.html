SHORT HAND-padding

  • untuk membuat ruang dikeliling konten elemen, letaknya didalam border, transparan
  • untuk ke4 nilai padding ingat searah jarum jam : atas, kanan, bawah, kiri
  • tidak boleh pakai nilai negatif spt -10px
  • value lain selain value shorthand :
    • tidak bisa pakai auto
    • inherit - buat elemen copy nilai padding elemen parentnya
  • nilai shorthand untuk : (tak apa2 jika salah satu nilai dibwh hilang, selama masih sesuai urutan)
    • padding-top
    • padding-right
    • padding-bottom
    • padding-left

    • aturan shorthand sama spt margin

  • cara pakai :

        body {
            padding: padding-top padding-right padding-bottom padding-left;
        }
