text-indent

  • digunakan untuk menentukan indentasi baris pertama dari teks
  • value :
    • length - mendefinisikan nilai tetap dalam satuan px, em, cm, pt, etc
    • % - mendefinisikan nilai dalam % dari elemen parentnya
    • initial - mengubahnya ke nilai default
    • inherit - mewarisi nilai property parent elemennya
  • cara pakai :

        body {
            text-indent: 10%;
        }
