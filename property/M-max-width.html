max-width

  • untuk meningkatkan penanganan browser jendela kecil. Ini penting saat berguna saat web dibuka dihp
  • valuenya :
    • none (default) - tak ada lebar max
    • length - nilai angka dlm satuan px, cm, em, dll
    • % - nilai angka dlm satuan %
    • initial - menjadikan lebar/tinggi ke nilai default
    • inherit - Tinggi / lebar akan diwarisi dari nilai induknya
  • cara pakai :

        div {
            max-width: 500px;
        }
contoh :
max-width : 500px;
maka saat dibuka dalam ukuran layar yg kurang dari 500px, width elemen tersebut akan mengecil dan nilainya sama dengan ukuran layar yg dipakai
