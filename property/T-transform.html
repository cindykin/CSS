transform

  • transformasi 2D atau 3D untuk elemen. Properti ini memungkinkan Anda untuk memutar, menskalakan, memindahkan, memiringkan, dll
  • width = sumbu x = tidur || height = sumbu y = tegak
  • value : (yg kek translateZ(z) x,y,z mereka bisa cuma kepanjangan klo ditaruh disini)
    • none - tidak ada transformasi
    • translate(x,y) - memindahkan elemen dari tempatnya (sesuai dengan parameter yang diberikan untuk sumbu X dan sumbu Y) (pergerakan bukan dari ujung browser, melainkan dari tempat awalnya)
    • rotate(angle) - memutar elemen searah jarum jam/berlawanan arah jarum jam (nilainya negatif) sesuai dengan derajat tertentu
    • scale(x,y) - menambah/mengurangi ukuran elemen (sesuai dengan parameter yang diberikan untuk width & height).
    • skew(x,y) - memiringkan elemen di sepanjang sumbu X dan Y dengan sudut yang diberikan
    • matrix(n,n,n,n,n,n) - menggabungkan semua metode transformasi 2D menjadi satu.

    • matrix3d(n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n)
    • rotate3d(x,y,z,angle) - memutar elemen searah jarum jam/berlawanan arah jarum jam
    • translate3d(x,y,z) - memindahkan elemen dari tempatnya
    • scale3d(x,y,z) - menambah/mengurangi ukuran elemen
    • perspective(n) - define a perspective view for 3D elemen

    • initial - mengubahnya ke nilai default
    • inherit - mewarisi nilai property parent elemennya
  • cara pakai :

        div {
            transform: translate(50px,100px);
        }
