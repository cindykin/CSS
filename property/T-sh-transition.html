SHORT HAND-transition

  • shorthand untuk : (tak apa2 jika salah satu nilai dibwh hilang, selama masih sesuai urutan)
    • transition-property - property yg ingin ditambah efek
    • transition-duration (required) - durasi transisi
    • transition-timing-function - menentukan kurva dari kecepatan efek transisi, memungkinkan efek transisi untuk mengubah kecepatan selama durasinya
    • transition-delay - menentukan kapan transisi dimulai
  • cara pakai :

        body {
            transition: width 2s linear 1s, height 4s(value 2 opsional);
        }
