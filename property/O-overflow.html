overflow

  • mengontrol apa yang terjadi pada konten yang lebih besar untuk daripada kontainernya
  • hanya bekerja utk block elemen dengan ketinggian tertentu
  • value :
    • visible (default) - kelebihan tak terpotong, konten yg berlebih tampil sampai diluar kontainer
    • hidden - kelebihan terpotong, sisa konten menjadi tak terlihat
    • scroll - kelebihan terpotong, tapi ditambahkan scrollbar utk melihat kelanjutan dari konten
    • auto - mirip seperti scroll, tapi menambahkan scrollbar hanya jika diperlukan
  • cara pakai :

        body {
            overflow: hidden;
        }
