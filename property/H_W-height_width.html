height width

  • gabungan width serta height disebut content haha
  • height, width hanya mengatur tinggi serta lebar content elemen tidak termasuk padding, border, atau margin
  • tidak boleh pakai nilai negatif spt -10px
  • valuenya :
    • auto (default) - Browser menghitung tinggi dan lebar
    • length - nilai angka dlm satuan px, cm, em, dll
    • % - nilai angka dlm satuan %
    • initial - menjadikan lebar/tinggi ke nilai default
    • inherit - Tinggi / lebar akan diwarisi dari nilai induknya
  • cara pakai :

        body {
            width: 500px;
            height: 500px;
        }
