@font-face

  • Font web memungkinkan desainer Web untuk menggunakan font yang tidak diinstal di komputer pengguna
  • cara pakai :

        @font-face {
            font-family: namabebas; //font kita akan dikenali sbg ini
            src : url(fonts/riot.ttf); //tempat font kita
        }
        * {
            font-family : namabebas;
        }
