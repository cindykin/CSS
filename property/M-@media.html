@media

  • digunakan dalam kueri media untuk menerapkan gaya yang berbeda untuk jenis / perangkat media yang berbeda.
  • berefek pas browser diresize sesuai ukuran yg sudah ditentukan
  • cara pakai :

        @media(max-width: 500px) {  // saat ukuran browser lebih kecil dari 500px background-color warna biru
            body {
                background-color: blue;
            }
        }
