text-transform

  • digunakan untuk mengontrol kapitalisasi teks
  • value :
    • none (default) - tak ada kapitalisasi, teks sama dengan yg diketik
    • capitalize - mengubah huruf pertama dalam setiap kata menjadi kapital
    • uppercase - mengubah semua kata jadi huruf besar
    • lowercase - mengubah semua kata jadi huruf kecil
    • initial - mengubahnya ke nilai default
    • inherit - mewarisi nilai property parent elemennya
  • cara pakai :

        body {
            text-transform: capitalize;
        }
