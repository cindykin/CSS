position

  • menetapkan jenis penentuan posisi yang digunakan untuk sebuah elemen
  • value :
    • static (default) - elemen diposisikan sesuai dgn aliran normal dokumen (property top, right, bottom, left, dan z-index tidak berpengaruh)
    • relative - posisi elemen tidak hilang, elemen jadi 2d

      • seluruh elemen 2d (property top, right, bottom, left, z-index bisa dipakai) top 0 posisinya dimulai dari ujung browser bukan dari elemen parentnya


    • absolute - posisi elemen hilang, elemen jadi 2d. Posisi elemen tidak bergantung pada posisi parent, melainkan pada halaman dokumen
    • fixed - Perilaku sama seperti absolute, selalu relatif terhadap dokumen, bukan parent tertentu, bedanya tidak terpengaruh oleh scroll

    • -webkit-sticky - Perilaku sama seperti relative, hingga lokasi scroll area pandang mencapai ambang yang ditentukan, di mana elemen mengambil posisi tetap yang diperintahkan untuk tetap
    • initial - mengubahnya ke nilai default
    • inherit - mewarisi nilai property parent elemennya, nilai posisi tidak menurun, jadi ini dapat digunakan untuk memaksanya secara khusus, dan mewarisi nilai pemosisian dari induknya
  • cara pakai :

        body {
            position: -webkit-sticky;
        }
